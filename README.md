# cornet3d
web interface to software defined radios

## Ports
Ubuntu 16.04

## Installing

This is a Web App that used a software build system that uses GNU make
files.  The make files (GNUmakefile) automatically download (via wget)
some javaScript libraries.

### To build

Run make:
```console
$ make
```
